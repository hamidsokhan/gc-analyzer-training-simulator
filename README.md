# GC Analyzer Training Simulator

An interactive engineering training tool for:

- Gas Chromatograph (GC) systems
- Calibration understanding
- Fault diagnosis
- Chromatogram interpretation

## Features

### Level 4 — Fault Diagnosis
- Sensor-based troubleshooting
- Alarm logic
- Engineering decision making

### Level 5 — Chromatogram Diagnosis
- Visual chromatogram analysis
- Root cause identification
- Calibration simulation
